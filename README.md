# Qt5Sample
QTでGUIアプリを開発。
## 環境
#### Ubuntu 16.04 desktop (ubuntu-ja-16.04-desktop-amd64.iso)
#### Qt 5.7.0 (qt-opensource-linux-x64-5.7.0.run)
これでQT Creatorを使ってQT4/QT5の開発が行えます。
